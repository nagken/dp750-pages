## Readiness checklist

- [ ] Workspaces, clusters, SQL warehouses, pools, access modes.
- [ ] Cluster policies, networking patterns (VNet injection, Private Link, NPI).
- [ ] Identity: SCIM from Entra ID, service principals (OAuth M2M), key vault-backed secret scopes.
- [ ] Unity Catalog hierarchy and grants.
- [ ] Storage credentials + external locations + managed vs external tables.
- [ ] Row filters, column masks, dynamic views.
- [ ] Lineage + system tables + Delta Sharing + Lakehouse Federation.
- [ ] Auto Loader, COPY INTO, Structured Streaming.
- [ ] Delta features: ACID, time travel, MERGE, OPTIMIZE, ZORDER, Liquid clustering, VACUUM, Predictive Optimization, CDF.
- [ ] Lakeflow Declarative Pipelines (DLT): expectations, modes, event log.
- [ ] Workflows: tasks, triggers (file arrival, continuous), notifications, conditional + for-each.
- [ ] Asset Bundles: databricks.yml, targets, deploy with service principal.
- [ ] Lakehouse Monitoring: profiles, dashboards.
- [ ] Cost levers: job clusters, spot, autoscaling, Photon, Predictive Optimization.
